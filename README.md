# Répertoire Scolaire Moussa Hamza

## 🚀 Déploiement

1. Upload ce fichier `index.html` sur ton repository GitHub
2. Upload tes PDF et images
3. Le site détecte automatiquement les fichiers et crée des dossiers quand il y en a plus de 3 !

## 📁 Fonctionnalités

- ✅ Détection automatique des fichiers via API GitHub
- ✅ Dossiers automatiques quand > 3 fichiers du même type
- ✅ Lecteur PDF intégré
- ✅ Visionneuse d'images
- ✅ Design responsive (mobile, tablette, ordinateur)

## 📤 Sur GitHub

1. Crée un repository
2. Upload `index.html`
3. Upload tes fichiers PDF et images
4. Active GitHub Pages (Settings → Pages → Deploy from branch)
5. C'est prêt !
