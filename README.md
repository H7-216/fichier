# Répertoire Scolaire - Détection Auto

## 🚀 Déploiement

1. Upload ce fichier `index.html` sur GitHub
2. Upload tes PDF et images
3. Le site détecte automatiquement les fichiers !

## 📤 Sur GitHub

1. Crée un repository
2. Upload `index.html`
3. Upload tes fichiers PDF et images
4. Active GitHub Pages (Settings → Pages → Deploy from branch)
5. C'est prêt !

Le site utilise l'API GitHub pour détecter automatiquement les fichiers.
